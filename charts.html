    Select the Langauge

    1. Click on"Business Setting"


    2. Click on "Currency"


    2. Click on "+" Button to Add Currency


    3. Add the Currency Details

    Follow these steps one by one

  • Write Currency Name.

  • Select the Country.

  • Select Currency Symbol Position Left(€10,$10) or Right(10PKR,10₹).

  • Write Currency Code/Symbol.

  • Write Currency Decimal Point.

  • Write Currency Thousand Point.

  • Write Currency Decimal Place.

  • Write Currency Exchange Rate.

  • Select Currency Activation Active or Inactive.

  • Set Currency as Default.

  • Submit the Currency Details.
